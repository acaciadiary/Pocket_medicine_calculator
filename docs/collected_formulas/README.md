## 💡 Clinical Guidelines
For all formulas:
- **Clinical judgment is paramount.** Scores are decision-support tools, not mandates.
- **Diuretic influence:** FENa is unreliable if diuretics were given within 12–24 hours; use FEUrea instead.
- **Correct weight usage:** For renal dosing (Cockcroft-Gault), determine if Actual, Ideal, or Adjusted body weight is appropriate.
- **Electrolyte correction rates:** Free water deficit correction must not exceed 8–10 mEq/L in 24 hours to avoid Central Pontine Myelinolysis.
